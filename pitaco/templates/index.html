Sobre

Este projeto começou na tentativa de aumentar as chances de ganhar na Megasena sem precisar participar de bolões, onde as chances são maiores, mas você precisa dividir o prêmio com outras pessoas. Este site é para quem quer ganhar o prêmio sozinho. :)

Atualmente estou usando 3 algoritmos:
1. Números mais frequentes;
2. Números há mais tempo sem sair;
3. Números pares e números ímpares
Se você tiver sugestão de novos algoritmos, me informe para melhorarmos ainda mais.
